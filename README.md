# После клонирования проекта выполнить

```
 $ cp .env.example .env
```

# и заполнить все пустые поля в файле .env

## В поле ALLOWED_HOSTS добавить локальный ipv4 адрес последним элементом

### Инициализация

```
 $ ./project.sh init
```

---

### Запуск

```
 $ ./project.sh start
```

---

### Удалить все

```
 $ ./project.sh drop
```
